# Sales-Data-Analysis-Project
-Developed an interactive Excel dashboard to analyze sales performance across multiple dimensions. The dashboard provides insights into total revenue ($61.5M+) and total sales (48K+), with dynamic filtering by sales channel (In-store and Online).

## Project Highlights
-Processed and analyzed 48K+ sales transactions, totaling $61.5M+ revenue.
-Developed interactive dashboards using Pivot Tables, Charts, and Slicers.
-Explored sales trends by category, country, payment method, and shipment provider.
-Visualized monthly revenue trends for performance monitoring.
-Cleaned and structured raw data for accurate reporting.

## Skills Demonstrated
-Data Cleaning & Preparation
-Exploratory Data Analysis (EDA)
-Data Visualization & Dashboarding
-Advanced Excel (Pivot Tables, Pivot Charts, Slicers, Lookup Functions)

## Usage
-Download 'sales_data_dashboard.xsxl'.
-open in Excel 2016 or later version.
-use 'SalesChannel' Slicer to view In-store vs Online sales.
-Interact with charts to explore revenue patterns and trends. 

## Learning Outcomes
-Hands-on experience creating **interactive Excel dashboards**.
-Practical understanding of **data cleaning, visualization, and storytelling**.
-Developed skills in summarizing large datasets into actionable insights.

## Project Outcome
Transformed unstructured sales data into a clear, interactive dashboard to support quick decision-making and insight generation.
